## 🚀 Proyectos Destacados

### 📊 Analisis de datos

* **[📊 Monitoreo emocional economico](https://github.com/FranciscoGG09/fintech-sentiment-risk-monitor.git)**
    > Desarrollo de un pipeline de datos end-to-end que monitorea la salud reputacional y el riesgo de mercado para instituciones financieras (bancos y cajas populares). El sistema procesa grandes volúmenes de texto no estructurado para detectar señales tempranas de crisis o cambios en la opinión pública sobre tasas de interés y préstamos. 

* **[📊 Prueba A/B](https://github.com/FranciscoGG09/ab-testing-conversion-optimization.git)**
    > Marco de trabajo experimental aplicado a una plataforma de servicios financieros para optimizar la tasa de conversión (CRO). El proyecto documenta el proceso científico desde la formulación de la hipótesis hasta la validación estadística de resultados en la adquisición de nuevos clientes de crédito. 

### 💻 Desarrollo web y movil

* **[🌟 Sistema Municipal de reportes](https://github.com/FranciscoGG09/SistemaMunicipalTancitaro.git)**
    > Sistema completo para la gestión y seguimiento de reportes ciudadanos y administrativos del municipio de Tancítaro. Consta de una **aplicación móvil** para captura de reportes (incidencias, servicios, quejas) y un **dashboard web** para su visualización, análisis y atención por parte del personal municipal.

* **[💈 App Web de Salón (AppSalon)](https://github.com/FranciscoGG09/AppSalon.git)**
    > Aplicación web Full Stack (PHP, JS, SQL) para la gestión de citas de salones de belleza/barberías. Incluye sistema de registro para clientes y panel administrativo.

* **[💻 Web de Conferencias (Web-Camp)](https://github.com/FranciscoGG09/Web-Camp.git)**
    > Sitio web de una conferencia (eventos) con registro de usuarios, manejo de agenda dinámica, y conexión con BD/pasarelas de pago.
